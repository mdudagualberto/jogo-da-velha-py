# ❌⭕ Jogo da Velha

> Jogo da velha para dois jogadores com tabuleiro numérico, rodando direto no terminal!

---

## 📖 Sobre o Projeto

**Jogo da Velha** é um jogo clássico para dois jogadores. O diferencial aqui é que o tabuleiro é composto por 9 números escolhidos pelos próprios jogadores no início da partida!

- O **Jogador 1** joga com `X`
- O **Jogador 2** joga com `O`
- Cada rodada, o jogador escolhe um número do tabuleiro para marcar sua posição
- Quem completar uma linha, coluna ou diagonal primeiro vence!

---

## 🎮 Como Funciona (Exemplo)

```
Digite 9 números para o tabuleiro: 1 2 3 4 5 6 7 8 9

Tabuleiro:
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9

Jogador X, escolha um número: 5

 1 | 2 | 3
-----------
 4 | X | 6
-----------
 7 | 8 | 9

Jogador O, escolha um número: 1

 O | 2 | 3
-----------
 4 | X | 6
-----------
 7 | 8 | 9

... e assim até alguém vencer! 🏆
```

## 🚀 Como Rodar

### Pré-requisitos

Você precisa ter o **Python** instalado na sua máquina.

> Não tem Python? Baixe em: https://www.python.org/downloads/

Para verificar se já está instalado, abra o terminal e digite:

```bash
python --version
```

Se aparecer algo como `Python 3.x.x`, está tudo certo!

---

### Passo a Passo

**1. Clone o repositório:**

```bash
git clone https://github.com/mdudagualberto/jogo-dav-velha-py.git
```

**2. Entre na pasta do projeto:**

```bash
cd jogo-da-velha-py
```

**3. Execute o jogo:**

```bash
python jogo_da_velha.py
```
